# Rudra Miyani | Personal Portfolio

Welcome to my personal portfolio repository! This project showcases my skills, projects, and professional background with a unique OS-style boot animation interface.

## 🚀 Features
- **OS-Style Welcome Screen:** Interactive terminal-like boot sequence (RUDRA_OS).
- **Responsive Design:** Optimized for all screen sizes (mobile, tablet, desktop).
- **Modern UI/UX:** Built with clean HTML, CSS, and Vanilla JavaScript.

## 📂 Project Structure
- `index.html`: Main structure of the portfolio.
- `style.css`: Stylesheet for layout, typography, and custom animations.
- `script.js`: Interactive elements including the boot sequence script.
- `images/`: Directory containing project assets and images.

## 💻 Tech Stack
- HTML5
- CSS3
- JavaScript
